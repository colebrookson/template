## what's this template for? 
A minimal reproducible template for my R projects that includes a local docker set-up and a protocol for adding dependencies. 

The container runs on a Linux distro from rocker, and is flexible to other platforms if the need arises. 

There is very little here by way of actual R code, only one file with a function in it that I like to use to make my ggplots prettier. Setting up the code for the actual project will depend more on the needs of an individual project, but the general basis of functions and scripts is probably the place to start, hence their sub-folders.
